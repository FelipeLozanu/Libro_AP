## Cómo está hecho este libro

Este sitio está escrito en archivos Markdown y notebooks de Jupyter, y se convierte a HTML usando herramientas del ecosistema [TeachBooks](https://teachbooks.io/) y [Jupyter Book](https://jupyterbook.org/).

Los archivos fuente están en un repositorio público de GitHub:

- https://github.com/elloza/teachbook_sciences_template

La versión publicada del libro puede consultarse en:

- https://elloza.com/teachbook_sciences_template/

## Cómo reproducir la web

Hay dos opciones principales:

1. **En GitHub**: hacer fork del repositorio, activar GitHub Pages con fuente `GitHub Actions` y ejecutar el workflow de despliegue.
2. **En local**: clonar el repositorio, instalar dependencias y compilar el libro con los scripts del proyecto.
