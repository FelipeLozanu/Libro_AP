# Assessment Rubric and Learning Objective Sequence

## Rubric: Research project

| Criterion | Excellent (4) | Good (3) | Sufficient (2) | Insufficient (1) |
|-----------|---------------|----------|----------------|-------------------|
| **Problem statement** | Defines an original, relevant problem with clear context | Defines an adequate problem with some context | Defines a vague or poorly contextualized problem | No identifiable problem defined |
| **Methodology** | Rigorous, well-justified, replicable method | Adequate method with partial justification | Simple method without clear justification | No defined method |
| **Data analysis** | In-depth analysis with correct interpretation | Correct analysis with basic interpretation | Superficial analysis with minor errors | No analysis or major errors |
| **Conclusions** | Well-founded conclusions answering the problem | Coherent conclusions aligned with data | Partial or poorly connected conclusions | No conclusions or unrelated to the work |
| **Presentation** | Impeccable structure, clear writing, no errors | Good structure, minimal errors | Acceptable structure, some errors | Disorganized, frequent errors |

```{admonition} Scoring
:class: tip
Minimum passing score: **10 / 20** (50 %). The total score is obtained by adding the values of each criterion.
```

## Learning objective sequence

| Phase | Objective | Expected outcome |
|-------|-----------|-----------------|
| 1. Exploration | Identify a topic of interest | List of 3 candidate topics |
| 2. Delimitation | Formulate a research question | Concrete, viable question |
| 3. Design | Select an appropriate methodology | Research protocol |
| 4. Collection | Gather relevant data | Organized dataset |
| 5. Analysis | Interpret the results | Findings report |
| 6. Communication | Present the final work | Written document + oral defense |

## Applications

- **Education**: formative and summative assessment of projects
- **Psychology**: designing criteria for case studies
- **Biology**: evaluation of lab practices

```{admonition} Adapt the rubric
:class: tip
Replace the criteria in the first column with those from your course and adjust the descriptors. The 1-4 scale is standard and easy to calibrate across evaluators.
```
