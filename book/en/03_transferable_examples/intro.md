# Transferable Examples

```{admonition} What are transferable examples?
:class: tip
They are code and analysis examples that **can be adapted to multiple disciplines** with minimal changes. The idea is simple: if you learn to plot a growth curve for Biology, the same technique works for modeling vocabulary acquisition in Language or disease spread in Medicine.
```

## Disciplines covered

| Example | Discipline | Core technique |
|---------|-----------|----------------|
| [Logistic growth](biology_logistic_growth.ipynb) | Biology | Differential equation modeling |
| [Likert questionnaire](psychology_likert_questionnaire.ipynb) | Psychology | Survey analysis |
| [Glossary and frequencies](language_glossary_frequency.md) | Language | Lexical frequency tables |
| [Assessment rubric](education_rubric_sequence.md) | Education | Assessment instrument design |
| [Simulated clinical data](medicine_simulated_data.ipynb) | Medicine | Descriptive statistics and correlation |
| [Mass balance](chemical_engineering_balance.md) | Chemical Engineering | Stoichiometric calculations |

```{admonition} Tip for teaching staff
:class: tip
Each example is designed so you can copy it, replace the data with your own, and use it directly in your course. No programming required: just substitute the values.
```
